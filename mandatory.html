Mandatory elements

These are the mandatory elements to include in the interface of Power App Canvas application for ESDC.

Mandatory elements for all institutions

  1. Topography
  2. Colours
  3. Header
  4. Footer
  5. Menus layouts

Topography

Main content area:

  • H1: Lato,22px, bold
  • H2: Lato, 18px, bold
  • H3: Lato, 16px, bold
  • Body: Open Sans, 14px, plain text
  • Links: Underline links in an underline style that skips descenders.

Header Full size layout:

  • Screen title H2 Lato,28, bold, #FFF
  • User Menu Lato, 16px, bold, #FFF

Header Condensed size layout:

  • Screen title H2 Lato,22, bold, #FFF
  • User Menu Lato, 14px, bold, #FFF

Footer Full size layout:

  • Text Lato, 16px, normal, #000
  • Links as default

Footer Condensed size layout:

  • Text Lato, 14px, normal, #000
  • Links as default

Menu:

  • Horizontal menu: text Lato, 14px, bold, #27437D
  • Horizontal menu hover/onfocus: text Lato, 14px, bold, underline: #27437D
  • Vertical menu links: text Lato, 14px, bold, #27437D
  • Vertical menu hover/onfocus: text Lato, 14px, bold, #27437D, background: #FFF

Colours

Background:

  • The default background colour for Canada.ca is white (#FFF).
  • While you may use another background colour for specific portions of the page, ensure that the majority of the page has a white background. When you use other background colours, ensure sufficient contrast with text to meet WCAG success criterion 1.4.6: Contrast (Enhanced) (Level AAA).

Text:

  • The text colour for Canada.ca is black (#000).

Always use the following colours for links:

  • default link: #0535d2
  • visited link: #7834bc

Additional colours:

Template:

  • Hover of header buttons #E6E6E6
  • Background of header area #26374a
  • Background of footer area and splash screen #FAFAFA

Only use these colours as accent colours:

  • main accent colour: #26374A
  • form error or required element indicator: #d3080c
